# **App Name**: Anabel Chura - Portfolio

## Core Features:

- Profile Display: Showcase Anabel's professional information, skills, and certifications.
- Experience Timeline: Display a timeline of professional experiences with roles and responsibilities.
- Blog Section: Display articles about agile methodologies, leadership, and metrics.
- Contact Information: Provide contact details like email, LinkedIn, and GitHub profiles.

## Style Guidelines:

- Primary color: Slate blue (#778899) to reflect professionalism and technical expertise.
- Background color: Off-white (#F8F8FF), subtly desaturated to highlight the content, provides a clean and modern feel.
- Accent color: Pale violet red (#D87093) used sparingly for interactive elements to bring subtle contrast and visual interest.
- Body text: 'Inter' (sans-serif) for a clean, readable, and modern appearance, ensuring excellent readability on all devices.
- Headline font: 'Space Grotesk' (sans-serif) for headings and titles, giving a technological and modern vibe.
- Use simple and professional icons related to skills, experience, and contact information, consistently styled.
- Layout with a header containing a profile photo, clear sections for experience, and a contact button to follow the given example.
- Subtle transitions and hover effects to enhance user experience.
